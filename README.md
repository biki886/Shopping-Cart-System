# Shopping Cart System

A simple **Shopping Cart System** built using **PHP, AJAX, Bootstrap, and MySQL**.
This project demonstrates how an e-commerce cart works with dynamic updates without reloading the page.

---

## Features

* Add products to cart
* Update product quantity
* Remove items from cart
* Dynamic cart updates using AJAX
* Responsive design using Bootstrap
* MySQL database integration

---

## Technologies Used

* PHP
* AJAX
* MySQL
* Bootstrap
* JavaScript
* HTML & CSS

---

## Installation

1. Clone the repository

```
git clone https://github.com/biki886/Shopping-Cart-System.git
```

2. Move the project folder to your server directory (XAMPP/WAMP)

```
htdocs/
```

3. Import the database file into **phpMyAdmin**

4. Start **Apache** and **MySQL**

5. Open the project in your browser

```
http://localhost/Shopping-Cart-System
```
